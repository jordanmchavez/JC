This document will contain the design for a simple space shooter game. The purpose of writing this game is as follows:
1. Ship a simple but complete game. My intention is to release this on Steam for a couple of dollars.
2. Build a minimalist 2d game framework alongside the game. I emphasize "framework" rather than "engine". This framework will be similar to Love2D in its functionality and simplicity.
3. Improve my game development skill.

I want to start with a very simple 2d, top-down space shooter ala gradius. I want the design to end up with a roguelike space shooter with meta progression and a zillion items that define each run ala binding of isaac.
